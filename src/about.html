This application named "E-Voting system usng blockchain" is developed by Team Script Monks,
the students of Global Academy of Technology, Bangalore - 560098.
The project aims to attract more voters to the voting process and make the voting more transparent.
A voting system that uses electronic means take care of casting and counting votes makes the process more reliable as there is no human intervension.
The frontend is developed usind HTML, CSS, Javascript and Bootstrap. The Backend is designed with a a simple smart contract that forms the main framork for this Dapp.
